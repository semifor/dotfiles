## Installation

    # clone the repository
    git clone git://github.com/semifor/dotfiles.git
    cd dotfiles
    git submodule update --init

    # create symlinks
    ln -s ~/dotfiles/inputrc ~/.inputrc
    ln -s ~/dotfiles/tmux.conf ~/.tmux.conf
    ln -s ~/dotfiles/vim ~/.vim
    ln -s ~/dotfiles/vimrc ~/.vimrc
